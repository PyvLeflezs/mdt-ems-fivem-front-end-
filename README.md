# MDT EMS - FiveM UI

## 🇫🇷 Description

Interface front-end d’un MDT (Mobile Data Terminal) pour EMS développé pour FiveM.  
Ce projet contient uniquement le design et l’interface utilisateur (HTML / CSS / JavaScript) et peut être intégré dans une resource FiveM en tant que NUI.

L’objectif est de fournir une base moderne et réaliste pour un système de gestion EMS (patients, interventions, rapports, etc.), entièrement personnalisable et adaptable à n’importe quel framework FiveM.

---

## 🇬🇧 Description

Front-end interface of an EMS MDT (Mobile Data Terminal) designed for FiveM.  
This project only includes the user interface (HTML / CSS / JavaScript) and can be integrated into a FiveM resource as a NUI.

The goal is to provide a modern and realistic base UI for an EMS management system (patients, interventions, reports, etc.), fully customizable and compatible with any FiveM framework.
